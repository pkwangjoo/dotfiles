EMACS CHEATSHEET
================

C = Ctrl, M = Alt/Option (or Esc then key)

MOVEMENT
--------
C-f         Forward one char
C-b         Backward one char
M-f         Forward one word
M-b         Backward one word
C-a         Beginning of line
C-e         End of line
C-n         Next line
C-p         Previous line
M-<         Beginning of buffer
M->         End of buffer
C-v         Scroll down (page down)
M-v         Scroll up (page up)
C-l         Center screen on cursor
M-g g       Go to line number

SELECTION (MARK/REGION)
-----------------------
C-SPC       Set mark (start selection)
C-x h       Select all
M-h         Select paragraph

CUT / COPY / PASTE
-------------------
C-w         Cut (kill region)
M-w         Copy (kill-ring-save)
C-y         Paste (yank)
M-y         Cycle through paste history (after C-y)
C-k         Kill from cursor to end of line
M-d         Kill word forward
M-DEL       Kill word backward

UNDO / REDO
------------
C-/         Undo
C-g C-/     Redo (undo the undo)

SEARCH / REPLACE
----------------
C-s         Incremental search forward
C-r         Incremental search backward
M-%         Query replace
C-M-%       Query replace (regex)

FILES / BUFFERS
---------------
C-x C-f     Open file
C-x C-s     Save file
C-x C-w     Save as
C-x b       Switch buffer
C-x k       Kill buffer
C-x C-b     List buffers

WINDOWS
-------
C-x 2       Split horizontal
C-x 3       Split vertical
C-x 1       Close all other windows
C-x 0       Close current window
C-x o       Switch to other window

TEXT EDITING
------------
C-d         Delete char forward
DEL         Delete char backward
M-u         Uppercase word
M-l         Lowercase word
M-c         Capitalize word
C-t         Transpose two chars
M-t         Transpose two words
C-o         Open new line below (without moving)
C-x C-t     Transpose two lines
M-q         Reflow/wrap paragraph

MISC
----
C-g         Cancel current command
C-x C-c     Quit Emacs
M-x         Run command by name
C-h k       Describe what a key does
C-h f       Describe a function

IVY / COUNSEL / SWIPER
----------------------
C-s         Swiper (search current buffer with preview)
M-x         counsel-M-x (fuzzy command search)
C-x C-f     counsel-find-file (fuzzy file open)
C-x b       ivy-switch-buffer (fuzzy buffer switch)
C-c C-r     ivy-resume (reopen last ivy session)
M-y         counsel-yank-pop (browse kill ring)

Inside ivy minibuffer:
  C-n         Next candidate
  C-p         Previous candidate
  RET         Select current candidate
  C-m         Same as RET
  TAB         Partial complete / show actions
  C-'         Use avy to jump to candidate
  M-o         Show extra actions for candidate
  C-c C-o     ivy-occur (send results to buffer)
  C-g         Cancel

PROJECTILE (project-scoped commands)
------------------------------------
C-c p f     Find file in project (fuzzy, like VS Code Ctrl+P)
C-c p s s   Search text in project (counsel-projectile-ag)
C-c p p     Switch to a known project
C-c p b     Switch buffer within project
C-c p d     Find directory in project
C-c p k     Kill all project buffers
C-c p R     Regenerate project file cache
C-c p !     Run shell command in project root
C-c p c     Compile project
C-c p T     Find test file in project

SHELL / TERMINAL
----------------
M-x eshell   Open eshell (Elisp shell — no PTY, no interactive CLIs)
M-x vterm    Open vterm (full terminal emulator — supports claude, htop, etc.)

  NOTE: eshell does NOT allocate a PTY, so interactive programs
  (claude, ssh, less, vim, etc.) won't work there. Use vterm instead.
